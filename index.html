Koinos Smart Contracts

This section contains documentation that's optimized for AI tools like large language models (LLMs). If you're using Cursor as your development environment, you can easily integrate this doc into your workflow.

💡 Best Practice: Cursor Integration

Just save your long-form smart contract documentation as a markdown file in your repository:


📄 Save your long doc as:

/docs/koinos-smart-contract.md

Then, configure .cursor.json to instruct the AI how to use it:


✍️ Add a rule in .cursor.json:

{
  "rules": [
    {
      "pattern": "docs/koinos-smart-contract.md",
      "instruction": "Use this file for understanding how Koinos smart contract implementation works."
    }
  ]
}

This setup ensures Cursor can assist you effectively based on your project-specific smart contract guide.
