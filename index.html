LLM Docs for Koinos Blockchain

Your AI‑optimized documentation hub for Koinos. Each page here is crafted to give large language models (LLMs) like Cursor or ChatGPT exactly the context they need to help you build on the chain.

How to Integrate with Cursor

1. Save your long‑form docs as Markdown in your repo, e.g.:

/docs/koinos-smart-contract.md

2. Add a rule in .cursor.json so Cursor knows how to use it:

{
  "rules": [
    {
      "pattern": "docs/koinos-smart-contract.md",
      "instruction": "Use this file for understanding how Koinos smart contract implementation works."
    }
  ]
}
